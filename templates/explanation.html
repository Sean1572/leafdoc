How does this all work?

Image taken from the TensorFlow website.

Using TensorFlow and the cumulation of SPIS skills and past programming experience, we struggled to create a model for classifying plants into either healthy or unhealthy categories.

Our original plan with Plant Checkup was to identify the health of a plant using image classification and machine learning. Our attempts to search the internet for datasets brought us to Google's Tensorflow datasets, which we merged based on the healthiness of the plant.

Loading in the datasets, we struggled to identify a working plan using the tensorflow_datasets library, Which pushed us to manually split the images two folders: healthy and unhealthy, with diseased plants falling under unhealthy. Our dataset consists of 4042 images.

Although we eventually had to accept that we couldn't understand every detail of Tensorflow's algorithm, we considered the whole library a black box which we can understand the function of the lines that we wrote. After surpassing this barrier of frustration, the rest of the project was considerably straight forward: save the model, and implement a user interface for someone to use in a real life situation. Using the backend Python Flask library and the Bootstrap CSS Library to set up and build the website, our product is far from finished, but is fully functional as of now.
